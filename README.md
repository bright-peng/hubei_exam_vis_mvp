# 湖北省公务员考试报名数据可视化平台

## 🌟 项目简介

这是一个专注于湖北省公务员考试（及其他类似公考）报名数据分析与可视化的开源工具。它能够帮助考生从宏观到微观全方位了解报名形势，做出更科学的报考决策。

**在线演示**: [访问 GitHub Pages](https://bright-peng.github.io/hubei_exam_vis_mvp/) (假设地址，请替换为实际地址)

## ✨ 核心功能

### 1. 📊 多维数据看板
- **实时数据总览**: 核心指标一目了然（总职位数、招录人数、当前报名人数、整体竞争比）。
- **动静结合**: 支持查看最新的静态数据快照，也可在本地环境进行动态数据管理。

### 2. 🎨 全新 UI 设计 (Arco Design)
- **现代化界面**: 采用字节跳动 Arco Design 设计语言，界面简洁、美观、操作流畅。
- **响应式布局**: 完美适配桌面端、平板和**移动端**，随时随地查看数据。
- **双版本支持**: 提供全新 UI 体验，同时保留经典怀旧版 UI，满足不同用户习惯。

### 3. 🗺️ 深度地图可视化
- **全省热力图**: 直观展示湖北省各市州的报名热度分布。
- **武汉专属视图**: 针对武汉市进行细粒度的**行政区级**下钻分析（如江岸区、武昌区、东湖高新区等）。
- **交互式体验**: 支持点击地图区域筛选职位、查看悬浮详情。

### 4. 📈 智能趋势分析
- **报名增长曲线**: 追踪每日报名人数变化，精准捕捉报名高峰期。
- **日增长榜单**: 自动识别单日报名增长最快（"飙升"）的岗位。
- **冷热职位榜**: 多维度展示竞争最激烈（"卷王"）和无人问津（"捡漏"）的岗位。

### 5. 🔍 强大的职位检索
- **多条件筛选**: 支持按工作地点（市/区）、学历、机关层级、关键词等组合过滤。
- **详细信息展示**: 
    - 包含**招录对象**、**研究生专业**、**本科专业**等关键字段。
    - 针对移动端优化的表格视图，重要信息不错过。
- **职位详情弹窗**: 提供便捷的详情查看模式，支持一键复制职位代码、收藏职位。
- **我的关注**: 允许用户收藏心仪岗位（Watch List），通过简单的职位代码即可批量追踪关注岗位的实时数据。

### 6. 🤖 全自动数据流
- **自动抓取**: 集成 Python 爬虫，自动从官方网站获取最新报名统计表。
- **自动处理**: 自动清洗数据、计算竞争比、生成分析报表。
- **自动部署**: 基于 GitHub Actions，每天定时（早/晚）自动更新数据并发布到网页，无需人工干预。

### 7. 📤 数据上传中心
- **可视化管理**: 提供独立的数据上传界面，支持用户手动上传 Excel 格式的**职位表**和**每日报名统计表**。
- **即时分析**: 上传后系统自动完成数据清洗、入库与指标计算，无需等待定时任务，即时更新看板数据。
- **智能校验**: 内置数据格式校验与引导，防止错误数据录入。

## 🛠️ 技术架构

### 前端 (Frontend)
- **框架**: React 18 + Vite 5 + **TypeScript**
- **UI 组件库**: Arco Design (主版) / Bootstrap (旧版)
- **可视化**: ECharts 5 (地图/图表) + Recharts
- **路由**: React Router 6

### 后端 (Backend & Data)
- **Web 框架**: FastAPI (用于本地开发和数据上传)
- **数据库**: SQLite (轻量级，适合单机/静态化场景)
- **数据处理**: Pandas (清洗、分析、格式转换)
- **爬虫**: Requests + BeautifulSoup4

### 部署 (Deployment)
- **静态化**: 前端构建为静态资源，数据预计算为 JSON。
- **托管**: GitHub Pages (免费、稳定、CDN 加速)。
- **CI/CD**: GitHub Actions 定时任务。

## 🚀 快速开始 (本地开发)

### 1. 环境准备
- Node.js 18+
- Python 3.9+

### 2. 后端启动 (数据处理服务)
```bash
cd backend
# 创建并激活虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 启动服务
python -m uvicorn main:app --reload
```

### 3. 前端启动 (界面交互)
```bash
cd frontend
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
前端访问地址: `http://localhost:5173`

## ⏰ 数据更新机制

本项目配置了 GitHub Actions 自动化工作流，按照以下北京时间定时运行：

| 时间 | 说明 |
|------|------|
| **06:00** | 早班车，获取隔夜后的最新统计 |
| **18:00** | 下班前，查看白天报名增长情况 |
| **22:00** | 晚间更新，获取全天最终数据 |

*注：如果官方未在对应时间发布新数据，系统会跳过更新。*
